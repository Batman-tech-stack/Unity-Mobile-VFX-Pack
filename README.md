# Unity Mobile VFX Pack

A collection of optimized visual effects created in **Unity using Shader Graph and VFX techniques**, designed specifically for **mobile games**.

## Features

* Mobile optimized VFX
* Shader Graph based effects
* Lightweight performance for Android devices
* Easy to integrate into Unity projects

## Effects Included

* Magic attack effect
* Fire effect
* Explosion effect
* Energy burst
* Hit impact VFX

## Tools Used

* Unity Engine
* Shader Graph
* URP (Universal Render Pipeline)

## Project Structure

Assets → Contains VFX assets and materials
Packages → Unity package dependencies
ProjectSettings → Unity project configuration

## How to Use

1. Clone the repository
2. Open the project in Unity
3. Open the scene containing the VFX examples
4. Drag and drop the effects into your game

## Purpose

This project is part of my **Unity portfolio** showcasing **mobile VFX optimization and Shader Graph skills**.
